Welcome to the WellConnect API Server

This server provides the backend API services for the WellConnect mobile application. It handles user authentication, well monitoring data, and push notifications.

API Documentation Send Push Notifications API Health Check

API Features

  • User Management: Registration, login, profile updates
  • Well Monitoring: Well data retrieval and management
  • Push Notifications: Device token registration for Firebase Cloud Messaging
  • Location Services: User location updates and nearby users

Getting Started

To interact with the API, you can use the following endpoints:

  • POST /api/register - Register a new user
  • POST /api/login - Login and get authentication token
  • GET /api/wells - Get all wells

For a complete list of available endpoints, visit the API Documentation.

Testing Tools

You can test the API using tools like:

  • The WellConnect mobile app
  • Postman
  • curl
  • Any HTTP client library
